Routers: IOS

Packet Tracer 5.1 uses a simplified model of the Cisco IOS. Click on the CLI tab in the router configuration window to access the Cisco IOS command line interface for the router. Use the Copy and Paste buttons to copy and paste text to or from the command line. This page lists the Cisco IOS command tree for the Cisco 2811 router which uses the superset command tree of all routers in Packet Tracer 5.1. The Cisco 2620XM/2621XM routers use a subset of this command tree. The tree contains only Cisco IOS command chains that are supported by the Packet Tracer 5.1.
